# Calculator
# Project Description
 This is a Command-Line Calculator built using Python.
 The calculator performs multiple mathematical operations including:
   1. ➕ Addition
   2. ➖ Subtraction
   3. ✖ multiplication
   4. ➗ division
   5. % percentage Calculation
      
      The program runs continuosly until the user chooses to exit.

# Features
1. User-friendly menu system 
2. Performs basic arithmetic operation 
3. Calculates percentage
4. Handles division by zero
5. Repeats program until user exits
6. Clean and structured code
